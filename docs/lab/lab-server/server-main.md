### 2.1 도메인

연구실 서버를 사용하기 위해서는 당연히 계정이 필요한데, 관리자에게 부탁하여 계정을 생성하자. 연구실 서버의 IP 주소는 `147.46.xx.xx`이며 `xxxx`번 포트를 이용하여 접속 가능하다.
